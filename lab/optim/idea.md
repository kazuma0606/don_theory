# lab/optim/idea.md — 最適化アルゴリズム開発アイデア

## 出発点となる問い

論文の実験では smooth あり/なしで劇的な差は認められなかった（収束率は同じ 80%）。
では、**Banach 空間の構造を直接利用することで初めて収束できる、あるいは明確に速いアルゴリズムは存在するか？**

ただし、その効果は**問題設定（非可換性の強さ・次元・介入数）に大きく依存する**可能性がある。
そのため、アルゴリズム開発と「効く問題クラスの特定」を並行して進める。

---

## O-0. 前提：効く条件の特定（全実験の基礎）

まず「どういう問題でSmoothing・Banach構造が効くか」を系統的に調べる。

**仮説**:
| 条件 | 予想効果 |
|---|---|
| 非可換性 `‖[A,B]‖` が大きい | 平滑化の恩恵が増大 |
| 介入数 K が多い | 景観が複雑化、Banach正則化が有効 |
| パラメータ次元 m が大きい（m ≫ 2） | Hessianが計算困難、一階法の差が出る |
| 初期値が悪い | 荒れた景観での差が顕在化 |
| 状態次元 d が高い | W^{1,∞} の安定化効果が拡大 |

**今回 m=2 だったため L-BFGS に負けた可能性**: m が大きくなるほど Hessian 計算は困難になり、Banach 空間の一階情報だけで収束するアルゴリズムの価値が出てくるはず。

---

## O-1. W^{1,∞} Mirror Descent

**アイデア**: 標準勾配降下は Euclidean 幾何を前提とする。
W^{1,∞} ノルムを Bregman ダイバージェンスの生成関数として使うことで、
「Lipschitz 幾何に沿った」更新則を導出する。

**更新則**:
```
θ_{t+1} = argmin_θ  <∇J_ε(θ_t), θ>  +  (1/η) B_φ(θ, θ_t)
```
ここで `B_φ` は W^{1,∞} ノルムから誘導される Bregman ダイバージェンス。

**期待効果**: 非可換作用素合成が誘発する非等方的な景観に対して、自然な探索方向を与える。

---

## O-2. 適応 ε スケジューリング

**アイデア**: 固定 ε ではなく、勾配ノルムの変動を見て ε を動的に下げる。
Corollary 2（ε → 0 で最小解が収束）を保証として組み込む。

**スケジュール案**:
- `‖∇J_ε‖` の変動（step-to-step ノルム差）が閾値以下になったら ε を半減
- 最終的に ε → 0 のときの解を参照解と比較

**検証点**: 固定 ε vs 適応 ε の収束ステップ数・最終誤差の比較。

---

## O-3. Frank-Wolfe in W^{1,∞}

**アイデア**: Frank-Wolfe（条件付き勾配法）は非 Euclidean 幾何で強みを持つ。
W^{1,∞} ボール `{f : ‖f‖_{W^{1,∞}} ≤ C}` 上の線形最小化オラクルを設計する。

**線形最小化オラクル**:
W^{1,∞} ボール上の線形最小化は、作用素の非可換構造から解析的に解ける可能性がある。

**期待効果**: 射影ステップが不要で、制約付き最適化に自然に対応。

---

## O-4. 非可換性ペナルティ付き SGD

**アイデア**: 損失関数に交換子ノルムを正則化項として追加し、景観を能動的に平滑化する。

**目的関数**:
```
L(θ) = J(θ) + λ · ‖[E_A(θ), E_B(θ)]‖_F
```

**意味**: 「非可換性を小さくしながら目標に近づく」という探索を促す。
介入順序に依存しない解（可換に近い解）への誘導効果を検証。

---

## O-5. Proximal-Mollifier 法

**アイデア**: W^{1,∞} ノルムを近接作用素として使う交互最小化。

**手順**:
1. `θ_{t+1/2} = argmin_θ  J(θ) + (1/2η)‖θ - θ_t‖²`（標準近接ステップ）
2. `θ_{t+1}   = prox_{W^{1,∞}}(θ_{t+1/2})`（W^{1,∞} 射影）

**期待効果**: mollifier の平滑化効果と近接法の収束保証を組み合わせる。

---

## O-6. 高次元パラメータ空間での検証（m ≫ 2）

**最重要実験**: 今回の論文は θ ∈ ℝ² だったため、Banach 空間の恩恵が小さかった可能性がある。

**実験設定**:
- θ ∈ ℝ^m、m = 2, 10, 50, 100, 500
- 各 m で Adam / L-BFGS / W^{1,∞}-Mirror Descent を比較
- 収束率・最終誤差・計算コストを記録

**仮説**: m が大きくなるほど Hessian 計算が困難になり、
W^{1,∞} + Mollifier を利用した一階法が相対的に優位になる。

---

## O-7. 収束保証付きアルゴリズムの理論設計

**長期目標**: 以下を満たすアルゴリズムを理論的に設計し数値検証する。

- **収束定理**: `J_ε(θ_t) - J_ε(θ*)  ≤  O(1/t)` の証明（凸近似の場合）
- **Banach 空間の Lipschitz 勾配** を陽に使った収束レート
- W^{2,∞} 仮定（Corollary 1 の前提）を緩和したアルゴリズム設計

---

1. **O-0**: 効く条件の特定（全実験の前提）
