# W-7 実験シリーズ — 考察

## 1. 主要発見の概観

本実験シリーズは，**介入操作の非可換性に起因するランドスケープ変形が，置換群 S_K の位相・代数的構造によって予測可能である**ことを系統的に実証した。
結論として，大 N スクリーニング（P@50）では **3-Stage-25%** が最良（P@50=0.868），
小 N スクリーニング（P@10, P@20）では **SRM+GF_M30** が最良（P@20=0.900）である。

---

## 2. なぜ Burau trace が機能するのか

Burau trace は組み紐群 B_K の unreduced 表現のトレースであり，置換が恒等に近いほど（Cayley 距離が小さいほど）大きな値を取る傾向がある。
ランドスケープ距離 dist は「参照順序との経路差」の RMSE であり，
恒等置換に近い順序ほど dist が小さい。
この幾何的な対応関係が r ≈ −0.53（K=6）の相関を生む。

ただし，相関が −1 でない理由は以下の 3 点にある：
1. **K-サイクルの逆転**：cycle type [6] では within-r = +0.124（逆向き），K-サイクルのべき乗が非単調ループを形成する
2. **cycle type 内の分散**：同じ cycle type（=同じ Cayley 距離）でも置換による軌跡への影響は異なる
3. **Jones 多項式の失敗**：K≥4 では Jones 多項式は dist との相関がほぼゼロ。Jones 多項式は knot/link の位相不変量として強力だが，K次元の積分性・非可換性には対応していない

---

## 3. スケーリング則の解釈

K の増加とともに |r| が弱まる（−0.83 → −0.53）のは，
置換空間 S_K が指数的に広がるにつれて Burau trace が捉えきれない **cycle type 内の分散**が増大するためである。
一方 p 値は劇的に小さくなる（n∝K! の増加）。

重要な示唆：**B₃（6置換）での高相関は部分的に偶然**であり，B₆（720置換）での中程度相関の方が普遍的な関係を反映している可能性がある。

---

## 4. 3-Stage-25% が機能する理由

3-Stage の強さの源泉は **Stage 0 の hard filter** にある。

25th パーセンタイル閾値でフィルタされる cycle type は
`{[1,1,1,1,1,1], [2,1,1,1,1], [3,1,1,1]}`（56/720 件，7.8%）である。
これらは Cayley 距離が 0, 1, 2 に対応する**「恒等元の近傍球」**であり，
この集合は S_K 上の Cayley グラフにおける半径2のボールに等しい。

つまり 3-Stage は：
> **「トポロジー的に恒等元に近い置換だけを候補に残す」**

という直観的に正しい操作を，データ駆動的に発見した形になっている。

この 56 件の中に真の上位 50 件のほとんどが含まれており（カバー率 ≈ 86.8%），
Stage 1/2 の within-type ランキングはむしろ副次的である。
実際，3-Stage+SRM でも 3-Stage+Jacobian でも P@50=0.868 と同一であり，
**「フィルタ後のランキング改善は天井がほぼ飽和している」**ことを示す。

P@50=0.868 の天井を破るには，現在フィルタで除外される 664/720 件の中にある「隠れた良い順序」を取り込むか，56 件の中でより精確なランキングをするかのどちらかが必要だが，前者は cycle type 構造と矛盾し，後者は R²≤0.27 の回帰天井に阻まれる。

---

## 5. なぜ機械学習は失敗するのか

全 ML 手法（SVM/RF/GBM/XGBoost/GP）が CV R²≤0.27 の天井を超えられない理由は：

1. **特徴量の情報量が飽和している**：Burau trace, Cayley 距離, cycle type ダミーが捕捉できる分散は全分散の約 27% であり，これは線形回帰でも同じ天井に達する。ML のノンパラメトリック性は追加情報を生まない。
2. **720 サンプルは複雑なモデルには小さすぎる**：XGBoost/GP のような高容量モデルは過学習し，交差検証で汎化しない
3. **ARD の崩壊**：ARD は多くの特徴量の係数を 0 に収縮させ，結果として Burau 単独と等価になる

**GaussianNB の逆説**：AUC=0.84（高い）だが P@50=0.57（低い）。NB は「良い順序か否かの二値分類」では良いが，「より良い順序を連続スコアで区別する」ランキングには不向き。これは課題が**ランキング問題**であることを示唆している。

---

## 6. Fourier 手法の解釈：なぜ小 N で優位か

SRM Fourier と Graph Fourier が **P@20 で 3-Stage を超える**（0.890, 0.900 vs ≈0.850）のは，以下の理由による：

### 6.1 SRM の意味

S_K の標準表現行列 M(σ) は，置換 σ がベクトル空間に作用する方法を符号化する。
25 次元 (K=6) の特徴ベクトルは「置換の群論的構造」をコンパクトに表現しており，
cycle type のような離散的分類よりも細かい情報を含む。

線形回帰 `dist ≈ w · srm(σ) + b` は **群論的線形予測**であり，
S_K の Fourier 展開の最低次近似と解釈できる。

### 6.2 Graph Fourier の意味

Cayley Laplacian の固有ベクトルは S_K 上の「調和関数」であり，
dist（距離関数）が Cayley グラフ上で滑らかであれば，
低周波固有ベクトル（小固有値）で well-approximated になる。
実際に M=50 が最適であり，それ以上のモード追加は雑音を増やす（過学習）。

### 6.3 小 N vs 大 N でのトレードオフ

- **小 N（@10, @20）**：上位 10–20 件は dist が非常に小さく，cycle type [1,1,1,1,1,1] の周辺に密集。この精密な差は Fourier の連続スコアが得意。
- **大 N（@50）**：上位 50 件には複数の cycle type が含まれ始める。3-Stage の「型ごとに 25th pctile 以下を全保持」というルールが，Fourier の単一線形予測より解像度が高い。

## 7. Wavelet_multi の失敗分析

Wavelet packet（複数スケール熱核特徴量の Ridge 結合）が P@50=0.116 と壊滅的な理由：

複数スケールの熱核 `exp(−s·λ)` フィルタは，スケール s が大きいほど低周波成分しか残らない。
各スケールのフィルタリング後の予測 `y_s(i) = U · diag(exp(−s·λ)) · (Uᵀ·y_tr)`（i∈全頂点）は，
異なる s でも非常に高い相関を持つ（多重共線性）。
Ridge 回帰が各スケールに大きく異なる係数を当てようとして不安定化し，
外挿で符号が逆転する。

これは **スケール間の情報冗長性** の問題であり，
直交化（PCA等）によって解決できる可能性があるが，本実験の範囲外。

## 8. E₁ 位置効果の発見（W-7m）

pos_E₁（E₁が順序中の何番目かを示す整数）と dist の相関は r = +0.502 であり，
単純な位置変数として Burau trace（r ≈ −0.53）に匹敵する予測力を持つ。

E₁ を最初に置いた場合 mean_dist=1.015，最後に置いた場合 mean_dist≈4.5 と，
**順序の最初の要素が dist を最も強く決定する**。

機構的解釈：E₁(p; θ) = p + θ₁·a_vec は **線形シフト演算子**であり，
p→p の Jacobian がほぼ恒等行列。最初に適用すると後続の非線形演算子の作用点が
参照順序とほぼ同じになるため，ランドスケープ変形が小さい。

この発見は実験設計上の直観的なルール「**E₁（線形シフト）を先に適用せよ**」を与える。

## 9. 実験計画への実用的示唆

### 状況別推奨手法

| 状況 | 推奨手法 | 根拠 |
|---|---|---|
| 720 通りから 10–20 通りに絞りたい | SRM+GF_M30 (P@20=0.900) | 小 N では Fourier 連続スコアが精密 |
| 720 通りから 50 通りに絞りたい | 3-Stage-25% (P@50=0.868) | 大 N では cycle type フィルタが支配的 |

1. **cycle type が [1,1,1,1,1,1], [2,1,1,1,1], [3,1,1,1] の順序を最優先する**
   → これだけで P@50≈0.87 が期待できる
2. **E₁（線形シフト）は順序の先頭に置く**
   → pos_E₁=0 の場合 mean_dist≈1.0（最小）
3. **SRM スコアが低い（≈0）順序は避ける**
   → 25 次元線形予測が補助的な絞り込みに使える

## 10. 理論的含意

### 10.1 S_K 調和解析との接続

SRM Fourier が機能することは，**dist が S_K 上の関数として S_K の表現論的構造に対して正則**であることを示唆する。具体的には：

```
dist(σ) ≈ Σ_ρ Σ_{i,j} c^ρ_{ij} · ρ(σ)_{ij}
```

ここで ρ は S_K の既約表現，c^ρ_{ij} は Fourier 係数。
SRM は非自明な既約表現（Standard Rep.）の行列要素を直接特徴量として使っており，
これは S_K 上の Fourier 解析の最低次近似に対応する。

### 10.2 Cayley グラフと Banach 空間の接続

本研究の理論枠組み（W^{1,∞} Sobolev 空間による滑らか化）と，
実験で発見した Cayley グラフ上の調和解析は，異なる側面から同じ現象を捉えている：

- **理論**：ランドスケープ関数 J(θ, E) を W^{1,∞} で近似し，ε-mollifier で滑らか化
- **実験**：dist(σ) を Cayley グラフ上の滑らかな関数として Laplacian 固有展開で近似

両者は「**非可換性が生む粗さ（roughness）を，適切な関数空間で滑らかにする**」という共通の操作原理を持つ。

### 10.3 開かれた問い

1. **r≤0.55 の天井は本質的か？**：Burau/SRM が捕捉できない dist の分散（73%）は何によるのか。高次表現を加えると改善されるか？
2. **K→∞ での挙動**：K が大きくなると Cayley グラフが展開し，Fourier 手法のスペクトルカットオフ M の最適値はどう変化するか？
3. **非交換子構造**：[E_i, E_j] = E_i∘E_j − E_j∘E_i の大きさが dist に直接相関する可能性。

---

## 11. 結論

W-7 シリーズを通じた主要な結論：

1. **Burau trace は有効だが不完全**：r≈−0.53 の相関は「組み紐位相不変量が介入順序の良さの代理指標になる」ことを示すが，単独では不十分
2. **3-Stage-25% は群論的に自然**：フィルタが Cayley ボール（恒等の近傍）を選択しており，理論的に well-motivated
3. **Fourier 手法は小 N で優位**：P@20=0.900 は実験計画の初期絞り込みに有用
4. **機械学習の限界は特徴量の情報量天井**：R²≤0.27 は Burau/SRM で表現できる分散の上限を反映
5. **E₁ 位置効果**：線形演算子を先頭に置くというシンプルなルールが単純な位置特徴量で捕捉可能

---

## 12. W-7-2 シリーズ：R²≈0.27 天井の解剖

W-7-2 では「なぜ ML 含む全手法が R²≤0.27 の天井を超えられないのか」という根本問いを，
表現論的階層（スカラー → 多点 → 固有値スペクトル → 忠実表現）に沿って系統的に検証した。

### 12.1 仮説の棄却：非忠実性は天井の原因ではない

W-7-2 以前の解釈では「Burau 表現が K≥5 で非忠実であるため情報を失っている」という仮説が自然だった。
これを直接検証した Phase 3 の結果は：

| 仮説 | 検証結果 |
|---|---|
| 「忠実な LKB を使えば R² が向上する」 | **棄却**: LKB trace の CV R² = 0.206 < Burau 0.272 |
| 「多点評価で情報量が増える」 | **棄却**: Burau 5点でも R² = 0.274 (+0.002 のみ) |
| 「固有値スペクトルで情報が増える」 | **棄却**: 固有値 6D でも R² ≤ 0.274 |

Burau の t=1/2 評価値という偶然の選択が「dist 予測に最適な scalar」に近かった，という解釈が最も整合的。

### 12.2 真の原因：スカラー圧縮

R²≈0.27 の天井は「表現の情報量（忠実性）」の問題ではなく，
**行列表現を 1 つのスカラー（トレース）に圧縮することによる構造的情報損失**が原因である。

- Burau 行列（K×K）をスカラー 1 個に圧縮 → R² = 0.272
- LKB 行列（15×15）をスカラー 1 個に圧縮 → R² = 0.206（むしろ悪化）
- SRM 行列（5×5）を **25 次元ベクトルとして保持** → R² = 0.934（突破）

表現行列の **行列要素をすべて特徴量として使う**（圧縮しない）ことが鍵であり，
LKB の 15×15=225 次元もすべて使えば同等以上の性能が期待できる（計算コストとの兼ね合い）。

### 12.3 LKB の逆説：なぜ忠実表現のスカラーが劣るのか

LKB trace が Burau trace より低い理由は表現の次元と評価点の選択の相互作用である：

- Burau (K=6): 6次元行列，t=1/2 での trace は概ね「固有値 {1, ζ₅, ζ₅², ζ₅³, ζ₅⁴, ...} の特定の重み付き和」
- LKB (K=6): 15次元行列，(x,y)=(1/2,1/2) での trace は高次の代数量であり，dist との幾何的対応が弱い

t=1/2 が Burau の「dist に最感応な方向」に偶然近かったのに対し，
(x,y)=(1/2,1/2) は LKB の最適評価点ではない可能性がある。
（評価点の系統的最適化は今後の課題。）

### 12.4 SRM のブレークスルー：R²=0.272 → 0.934

SRM（標準表現行列）の 25 次元特徴量は W-7m から既知だったが，
W-7-2 がはじめて「SRM が R² 改善の本質的な担い手」であることを確定した：

- LKB なし，SRM あり: R² = 0.930（W-7m の SRM_linear と同等）
- LKB あり，SRM なし: R² = 0.275（LKB の貢献はほぼゼロ）
- LKB あり，SRM あり: R² = 0.934（SRM に LKB がわずかに上乗せ）

SRM が機能する理由：SRM の (5×5) 行列要素は S_K の標準既約表現の行列要素であり，
`dist(σ) ≈ Σ_{i,j} c_{ij} · SRM(σ)_{ij}` という線形近似の基底として，
dist の S_K 上の変動パターンを効率よくカバーする。

### 12.5 改訂された情報階層

```
[忠実性の軸]     LKB > Burau            → R² 差: LKB 0.206 < Burau 0.272
[圧縮量の軸]     スカラー < 行列全体     → R² 差: trace 0.272 << SRM_25D 0.930
[次元の軸]       1D < 6D < 15D < 25D    → SRM_25D が最効率
```

情報量の天井は表現の「位相的情報（忠実/非忠実）」ではなく，
「表現の使い方（スカラー圧縮 vs 行列展開）」が支配していた。

### 12.6 開かれた問い（改訂）

1. **LKB 行列要素全展開**：LKB 行列の 225 要素をすべて特徴量に使えば R² はどこまで上がるか？（推定 ≥ 0.95）
2. **LKB の最適評価点**：(x,y) をグリッド探索して「dist に最感応な評価点」を発見できるか？
3. **SRM R²=0.93 の残余分散**：dist の 7% はどのような置換・演算子特性で説明されるか？
4. **K→∞ でのスケーリング**：SRM 次元 (K−1)² ∝ K² が大きくなったとき R² はどう変化するか？
5. **高次既約表現**：S_K の他の既約表現（外積表現など）を追加すると R² が飽和するか？
