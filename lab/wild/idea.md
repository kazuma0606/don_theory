## W-1. 経路空間とトポロジー

### アイデア

介入列 `(I_{σ(1)}, I_{σ(2)}, ..., I_{σ(K)})` は「経路空間（path space）」の点として見なせる。
σ は介入の順序置換。

- K 個の介入があれば K! 通りの経路が存在する
- 非可換性のせいで、経路が少し変わると出力 `f_{E_θ}` が大きく変わる
  → **経路空間のトポロジーが「荒い」**
- mollifier smoothing は、経路の違いに対する応答を連続的にする
  → **経路空間に滑らかな構造（接続・曲率）を入れる操作**

### 具体的な問い

- K=3 のとき 6 通りの経路がある。各経路での `J(θ)` を比較すると、景観はどう違うか？
- 経路間の「距離」を `‖f_{E_σ} - f_{E_τ}‖_{W^{1,∞}}` で定義したとき、この距離空間はどんな幾何をもつか？
- smoothing は経路空間の距離を縮めるか？（近い経路の出力が近くなるか）

### 接続する数学

- **基本群（π₁）**: 経路の「ループ」が存在するか？介入を元に戻す経路があるか？
- **Groupoid**: 群の一般化。部分的にしか合成できない変換の代数 → 介入列の自然な代数構造
- **Cobordism**: 経路空間のトポロジー的変形

### 実験 `exp_W1.py`

- K=3, d=64 で 6 通りの順序すべてで `J(θ)` を計算し、landscape を並べて可視化
- 経路間 W^{1,∞} 距離行列を計算し、MDS（多次元尺度法）で経路空間を 2D に埋め込む
- smoothing 前後で経路空間の「距離行列」がどう変わるかを比較

## W-2. バタフライエフェクトと ε の制御

### アイデア

非可換な世界では介入の順序をわずかに変えるだけで結果が大きく変わる
→ **バタフライエフェクト的な敏感依存**

ε はそのバタフライを「どの解像度まで見るか」を制御するパラメータ：

- ε 大：高周波バタフライを全部削除。大域的な分岐構造だけ残る
- ε 小：細かいバタフライまで見える
- ε → 0：元の荒れた landscape に戻る

**「どのレベルのバタフライを残し、どのレベルを消すか」を ε で制御する**

### 非可換性 Lyapunov 指数（仮称）

通常の力学系では Lyapunov 指数が「初期条件への敏感性」を測る。
ここでは類似物として：

```
λ_nc = lim_{δ→0} (1/K) log(‖f_{E_σ}(θ) - f_{E_τ}(θ)‖ / ‖σ - τ‖)
```

σ, τ が近い順序置換に対して、出力の乖離がどのレートで拡大するか。

### 実験 `exp_W2.py`

- 介入順序のランダムな微小摂動（σ → σ + δ）に対して出力の変化を計測
- ε に対して「バタフライ感度」がどう変わるか（ε が大きいほど感度が落ちるはず）
- λ_nc の数値推定と、非可換子 `‖[E_A, E_B]‖` との相関

## W-3. 帯域制限としての Mollifier

### アイデア

信号処理の「帯域制限（band-limiting）」と mollifier smoothing の対応：

| 信号処理 | この論文 |
|---|---|
| 元の信号 | `J(θ)`（ギザギザの landscape） |
| 高周波成分 | 非可換合成が作る局所的な深い谷 |
| 低周波成分 | 大域的な構造（真の最小値の近傍） |
| ローパスフィルタ | mollifier `ρ_ε` |
| 帯域幅 | `1/ε`（ε 大 → 帯域狭い → 強い平滑化） |

**実験で見えた「帯域外の深い谷」**：
- raw `J` で loss が `10^{-16}` まで落ちたのは「帯域外の高周波ノイズが作る偽の谷」
- smooth `J_ε` の loss フロア `10^{-2}` は「帯域制限された丸められた谷」

→ **smoothing は「良い解を保証する魔法」でなく「偽の谷を消すフィルタ」**

### フーリエ解釈

- パラメータ空間 Θ 上でフーリエ変換を取ると：
  `J_ε_hat(ξ) = J_hat(ξ) · ρ_ε_hat(ξ)`
- ρ_ε_hat は高周波 ξ で急速に 0 に近づく（ガウシアンなら指数的減衰）
- つまり `J_ε` は `J` の低周波成分だけを保持している

### 実験 `exp_W3.py`

- `J(θ)` の 2D フーリエ変換を計算し、パワースペクトラムを可視化
- mollifier 適用後 `J_ε(θ)` のパワースペクトラムと比較
- 「削られた高周波成分」の空間分布を可視化（どこが「偽の谷」か）

## W-4. LoG（Laplacian of Gaussian）による非可換性の可視化

### アイデア

画像処理の LoG（Laplacian of Gaussian）:
```
LoG(f) = ∇²(G_σ * f) = (∇²G_σ) * f
```
→ Gaussian でぼかしてから Laplacian を取る = エッジ（急峻な変化点）を検出

この論文の枠組みでは：
```
LoG_ε(J)(θ) = ∇²_θ J_ε(θ) = ∇²_θ (J * ρ_ε)(θ)
```

- `J_ε` が C^∞ なので ∇²_θ は完全に well-defined
- `LoG_ε(J)` は「ε スケールの曲率」を可視化する
- **これは非可換合成が作る景観の「急峻な分岐点」を抽出する**

### 解釈

- `LoG_ε(J)(θ) > 0`：θ 付近が谷（局所最小値）
- `LoG_ε(J)(θ) < 0`：θ 付近が山（鞍点・局所最大値）
- `LoG_ε(J)` のゼロ交差：「真の構造と偽の谷の境界」

ε を変えながら LoG を比較すると：
→ **どのスケールの構造が非可換性由来かを特定できる**

### 実験 `exp_W4.py`

- 2D parameter grid 上で `J(θ)`, `J_ε(θ)`, `LoG_ε(J)(θ)` を計算・可視化
- ε = 2.0, 1.0, 0.5, 0.1 で LoG の変化を追う（スケールスペース）
- LoG のゼロ交差と「偽の谷の位置」を重ねて表示

## W-5. d'Alembertian と介入の「波」

### アイデア

`J_ε ∈ C^∞` なので、あらゆる偏微分作用素が適用できる。
d'Alembertian（波動方程式の作用素）:

```
□ = ∂²/∂ε² - c² ∇²_θ
```

ここで ε を「時間的」パラメータ、θ を「空間的」パラメータと見なす。

`□ J_ε = 0` は「景観が波として伝播する」という解釈を与える：
- ε を大きくする（平滑化を強める）と、局所的な谷が消えて「波が伝播するように」広がる
- 介入の影響が θ 空間を波として伝播する

**面白い問い**：
- `□ J_ε` は実際にどんな値を取るか？
- 波動方程式 `□ J_ε = 0` を満たす (ε, θ) の領域はどこか？
- 「介入の波の速度 c」に物理的意味はつけられるか？

### 実験 `exp_W5.py`

- `□ J_ε` を数値微分で計算し可視化
- ε を連続的に変化させたときの `J_ε(θ)` の変化を「波の伝播」として動画化
- `□ J_ε ≈ 0` になる領域の特定

## W-6. 持続ホモロジー（Persistent Homology）による景観解析

### アイデア

位相的データ解析（TDA）の持続ホモロジーを使って：
- `J(θ)` と `J_ε(θ)` の「トポロジー的特徴」を比較する

持続ホモロジー：
- 損失景観の sublevel set `{θ: J(θ) ≤ c}` の c を 0 から ∞ へ動かす
- connected component（β₀）、ループ（β₁）の誕生・消滅を追う
- **「どの谷が本物でどの谷が非可換ノイズか」をバーコードで可視化**

**仮説**：
- raw `J` のバーコード：短命なトポロジー的特徴が多い（非可換ノイズが作る偽の谷）
- smooth `J_ε` のバーコード：長命な特徴だけ残る（真の構造）
- ε を大きくするほど短命な特徴が消えていく

### 実験 `exp_W6.py`

- `giotto-tda` または `ripser` を使って持続ダイアグラムを計算
- raw vs smooth の持続バーコードを比較
- ε に対して「短命な特徴の数」がどう減るかをプロット

## W-7. トポロジーとの接合（最もぶっ飛んだ方向）

### アイデア

非可換な介入列は **braid（編み紐）** として表現できるかもしれない。

- K 個の介入がそれぞれ「時間軸上の紐」
- 非可換性 = 紐の交差（どちらが上か）
- 異なる順序の介入列 = 異なる braid
- **Braid group（編み紐群）B_K が自然な代数構造として出てくる**

Braid group の性質：
- 非可換群（K ≥ 3 で）
- 位相不変量と深く関連（Jones 多項式 → 量子力学）

**面白い問い**：
- 介入列の効果（`f_{E_θ}` の値）は braid の位相不変量で分類できるか？
- Braid group 上に W^{1,∞} ノルムを定義できるか？
- smoothing は braid のどの位相不変量を保存し、どれを壊すか？

### 接続する数学

- **Braid group** → 非可換群の最も自然な例の一つ
- **Yang-Baxter 方程式** → 「交差の順序によらない」条件 ≈ 可換性
- **量子群** → 非可換幾何の代数的枠組み（Connes の非可換幾何学との接続）

### 実験 `exp_W7.py`（最も投機的）

- K=3 の介入を braid として表現（3 本の紐、交差のパターン）
- 各 braid に対して `J(θ)` を計算し、「位相的に同値な braid は同じ loss を持つか？」を検証
- 位相的に非自明な braid（Hopf link 的な構造）が landscape に与える影響を観察

## W-7-2. 多変量トポロジー不変量による dist 予測の天井探索

### 背景

W-7 シリーズで Burau trace（スカラー 1 個）の相関が r ≈ −0.53，
SRM 線形回帰（25 次元）を使っても CV R² ≈ 0.27 の天井が破れなかった。
この天井が「Burau の情報不足」によるものか，
「代数的不変量では原理的に予測できない dist の変動が 73% ある」のかを判別したい。

### アイデア

Burau 表現は B_K → GL(K, ℤ[t, t⁻¹]) の準同型だが，K≥5 で非忠実（核が非自明）。
つまり同じ Burau 行列を持つ異なる組み紐が存在する。
この情報損失の階層：

```
組み紐 b ∈ B_K
  ↓ Burau 行列（K×K，ℤ[t,t⁻¹]）← 非忠実（K≥5）
  ↓ t=1/2 で評価（実数行列）
  ↓ trace（スカラー）← W-7 はここだけ
```

これを段階的に「解凍」していくことで，どの層が dist に効いているかを特定できる。

### 特徴量の候補

| 特徴量 | 次元 | 意味 |
|---|---|---|
| `burau(1/2)` | 1 | W-7 の基準スカラー |
| `burau(t)` at t∈{1/3, 1/4, 2/3, 3/4} | 4 | 同じ表現の別の切り口 |
| `eigenvalues of Burau(1/2)` | K−1 | trace より豊富なスペクトル情報 |
| writhe | 1 | 交差符号の和（組み紐語に依存） |
| word length（既約語の長さ） | 1 | Cayley 距離の組み紐版 |
| SRM entries | (K−1)² | S_K 標準表現（W-7m で有効） |
| LKB trace / 固有値 | K(K−1)/2 | **忠実表現**（B_K を完全識別） |

### 実験設計

**Phase 1（低コスト）**：Burau(t) を複数点で評価した特徴ベクトルで R² が上がるか検証
```python
t_vals = [1/4, 1/3, 1/2, 2/3, 3/4]
features = [float(b.burau_matrix()(t=QQ(p)/QQ(q)).trace()) for p,q in ...]
```
→ R² が 0.27 から有意に上昇すれば「Burau の単一評価点が情報ボトルネック」
→ 上昇しなければ Phase 2 へ

**Phase 2（中コスト）**：Burau の固有値スペクトル（K−1 個）を追加
→ trace と eigenvalues は非線形な関係。Burau(1/2) が同じでも固有値分布が異なるケースを識別

**Phase 3（高コスト）**：LKB 表現（忠実）の trace または固有値を追加
→ LKB は SageMath で `b.burau_matrix()` の拡張として計算可能か要確認
→ 忠実なので「同じ LKB 値 → 同じ dist」が成り立てば，残差は pure dynamical noise

### 交差検証設定

W-7j 以降と同一プロトコルで比較可能にする：

```python
from sklearn.model_selection import KFold, cross_val_score
kf = KFold(n_splits=5, shuffle=True, random_state=int(42))
```

- **K=6，720 件，5-fold CV**（各 fold: train=576, test=144）
- 評価指標：CV R²（回帰精度）と Precision@N（N=10,20,50 のスクリーニング性能）
- 各 Phase で同じ fold 分割を使い，Phase 間の性能差を直接比較

### 解析モデル

#### A. Lasso / Ridge（特徴量選択 + 正則化回帰）

```python
from sklearn.linear_model import LassoCV, RidgeCV
# alpha を CV で自動選択
lasso = LassoCV(cv=5, random_state=int(42))
ridge = RidgeCV(alphas=[0.01, 0.1, 1, 10, 100], cv=5)
```

- `lasso_path` で各特徴量の寄与の大きさと安定性を可視化（どの不変量が有効かを判断）
- Lasso の零化された係数 = 「dist に効かない不変量」の特定

#### B. 重回帰分析（OLS + 有意性検定）

```python
import statsmodels.api as sm
X_with_const = sm.add_constant(X_features)
model = sm.OLS(y_dist, X_with_const).fit()
print(model.summary())  # t値・p値・VIF・調整済み R²
```

- **各不変量の回帰係数・t 値・p 値**を出力し，統計的有意性を確認
- **VIF（分散拡大因子）**で多重共線性を診断（Burau 複数評価点は高相関になりやすい）
- 調整済み R²（Adjusted R²）で特徴量追加の実質的な寄与を評価
- CV との照合：OLS の p 値が有意でも CV R² が上がらなければ過学習のシグナル

```python
from statsmodels.stats.outliers_influence import variance_inflation_factor
vif = [variance_inflation_factor(X.values, i) for i in range(X.shape[1])]
```

### 検証指標

- CV R² の天井がどの Phase で変化するか（または変化しないか）
- OLS の調整済み R² と CV R² の乖離（過学習の検出）
- 各特徴量の Lasso 係数の安定性（fold 間の標準偏差）
- Precision@50 が 3-Stage-25%（0.868）を超えるか

### 理論的含意

- **天井が破れる場合**：dist は B_K の忠実表現で完全に予測可能 → 組み紐の位相が実験ランドスケープを決定する
- **天井が破れない場合**：残差 73% は演算子の dynamical 情報（Jacobian 等）に依存しており，代数不変量だけでは原理的に届かない → 「Burau が捉える部分と dynamical 部分の直交分解」という新しい問いが立つ
- **VIF が高い場合**：Burau 複数評価点は線形従属に近く，主成分分析（PCA）で直交化してから回帰する方が安定
